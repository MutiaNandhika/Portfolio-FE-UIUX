## Tentang Proyek

**GoldenActivities** adalah platform berbasis web yang dirancang untuk mendukung keterlibatan lansia dalam aktivitas sosial dan kebugaran. Aplikasi ini dikembangkan untuk **kompetisi Web Development Soedirman Technoporia 2024**, dengan fokus pada inklusi digital dan peningkatan kualitas hidup bagi kelompok usia lanjut.

📍 Proyek Kompetisi: **Soedirman Technoporia 2024**  
🏫 Universitas Jenderal Soedirman – Teknik Informatika  
📅 Tahun: 2024  
🎯 Slogan: *Menanam Semangat Untuk Masyarakat*

## Fitur Utama

- **Beranda**: Informasi umum, pengantar, dan pembaruan
- **Aktivitas**: Daftar kegiatan sosial dan fisik untuk lansia
- **Pencapaian**: Sistem penghargaan untuk partisipasi aktif
- **Artikel**: Tips dan edukasi kesehatan lansia
- **Profil Pengguna**: Ubah data pribadi dan preferensi
- **Admin Panel**: Kelola pengguna, artikel, aktivitas, dan pencapaian

## Teknologi

- **Frontend**: HTML, CSS, JavaScript (Blade template engine)
- **Backend**: Laravel 11 (PHP)
- **Database**: MySQL

## Pengembangan & Timeline

Metodologi: iteratif dengan pendekatan **user-centered design**  
Tools desain awal: **Figma**  
Testing: Blackbox
