## Tentang Aplikasi

**SIREK (Sistem Informasi Rekrutmen)** adalah aplikasi mobile yang dikembangkan untuk memfasilitasi proses open recruitment kepanitiaan BEM Unsoed secara digital. Aplikasi ini bertujuan menyederhanakan proses perekrutan dan meningkatkan partisipasi mahasiswa melalui sistem yang terstruktur dan efisien.

## Tujuan

🎯 Mempermudah pengelolaan pendaftaran panitia  
🎯 Meningkatkan partisipasi aktif mahasiswa  
🎯 Menyediakan sistem informasi organisasi berbasis mobile  

## Fitur Utama

### 👨‍🎓 Untuk Mahasiswa
- Melihat daftar event kepanitiaan aktif
- Mendaftar secara online dengan formulir + unggah CV & LoC
- Unduh hasil pengumuman dalam format PDF

### 🧑‍💼 Untuk Admin & Pimpinan BEM
- Login aman
- Tambah/Edit/Hapus event dan pengumuman
- Manajemen data pendaftar lengkap

## Teknologi

- **Flutter**: Pengembangan aplikasi mobile multiplatform  
- **Firebase**: Autentikasi, Firestore DB, Storage  
- **Figma**: Perancangan antarmuka UI/UX  
- **Dependencies**: `firebase_auth`, `cloud_firestore`, `firebase_storage`, `dio`, `intl`, `file_picker`

## Metodologi Pengembangan

- 📌 **Model SDLC Waterfall**:
  1. Analisis kebutuhan
  2. Desain sistem (UML: use case, class diagram, dll.)
  3. Implementasi dengan Flutter + Firebase
  4. Pengujian (Blackbox)
  5. Pemeliharaan

## Hasil Pengujian

✅ Semua fitur berhasil diuji menggunakan metode blackbox  
✅ Fungsi login, CRUD event, pendaftaran, dan pengumuman bekerja sesuai rencana
