## Apa itu Info Berkoh?

**Info Berkoh** adalah platform website kelurahan berbasis PHP yang dikembangkan untuk menyajikan informasi publik secara komprehensif. Website ini berisi data penduduk, layanan masyarakat, berita terkini, pelaporan pengaduan, dan dokumentasi visual Kelurahan Berkoh.

## Teknologi yang Digunakan

- `PHP` (v8.x) untuk backend
- `MySQL` untuk penyimpanan data
- `CSS` untuk styling halaman
- Arsitektur CRUD (Create, Read, Update, Delete)
- Konfigurasi dan testing dilakukan melalui `XAMPP` dan `phpMyAdmin`
