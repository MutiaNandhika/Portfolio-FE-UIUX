## Tentang Proyek

**SIREK Web** adalah sistem informasi berbasis Laravel dan Vue yang dirancang untuk mendukung proses perekrutan panitia kegiatan oleh BEM Universitas Jenderal Soedirman. Platform ini memungkinkan interaksi tiga pihak utama—admin, pimpinan BEM, dan mahasiswa—dalam satu sistem terpadu yang efisien, transparan, dan real-time.

📍 Proyek: Implementasi Mata Kuliah Rekayasa Perangkat Lunak  
🏫 Universitas Jenderal Soedirman – Teknik Informatika  
📅 Tahun: 2024

## Fitur Utama

- **Administrator Website**  
  - Login aman dan dashboard
  - CRUD data event, pendaftar, pengumuman
  - Unggah dan kelola hasil seleksi

- **Pimpinan BEM**  
  - Akses penuh terhadap seluruh data event
  - Tambah/edit/hapus event & pengumuman
  - Pantau pendaftar dan status seleksi

- **Mahasiswa Unsoed**  
  - Lihat informasi event BEM
  - Isi formulir pendaftaran
  - Unduh hasil seleksi secara langsung

## Teknologi yang Digunakan

- **Backend**: Laravel 10 (PHP)
- **Frontend**: Inertia.js + Vue.js
- **Styling**: Tailwind CSS
- **Database**: MySQL
- **Deployment**: Nginx (VPS)
