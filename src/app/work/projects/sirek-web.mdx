## Fitur Utama

- **Administrator Website**  
  - Login aman dan dashboard
  - CRUD data event, pendaftar, pengumuman
  - Unggah dan kelola hasil seleksi

- **Pimpinan BEM**  
  - Akses penuh terhadap seluruh data event
  - Tambah/edit/hapus event & pengumuman
  - Pantau pendaftar dan status seleksi

- **Mahasiswa Unsoed**  
  - Lihat informasi event BEM
  - Isi formulir pendaftaran
  - Unduh hasil seleksi secara langsung

## Teknologi yang Digunakan

- **Backend**: Laravel 10 (PHP)
- **Frontend**: Inertia.js + Vue.js
- **Styling**: Tailwind CSS
- **Database**: MySQL
- **Deployment**: Nginx (VPS)
