## 🎨 Selected UI/UX Projects

### 1. Go Liboor – Mobile Travel Discovery App

A mobile app prototype designed to help travelers find destinations that match their interests. Built with a focus on clean navigation and interest-based filtering.

![Go Liboor preview](/images/uiux/go-liboor.png)

### 2. Inventory Management App (External)

Designed for PT Kilang Pertamina Internasional to improve warehouse coordination. The UI/UX prototype solves inefficiencies by digitizing approvals and logistics tracking.

![Inventory App preview](/images/uiux/inventory-pertamina.png)

### 3. Info Berkoh – Local Info Website

A community-based web platform providing demographic data, infographics, and neighborhood news for Kelurahan Berkoh.

![Info Berkoh preview](/images/uiux/info-berkoh.png)

### 4. SoalPedia – Educational Question Bank

An interface for an online question bank app that supports learning and assessment. Features a structured layout to categorize and retrieve question types easily.

![SoalPedia preview](/images/uiux/soalpedia.png)

### 5. Nucifero – Flower Bouquet E-Commerce

A web UI project for bouquet customization and ordering. Optimized for mobile use with a clean catalog and easy checkout.

![Nucifero preview](/images/uiux/nucifero.png)

### 6. Flutix App – Cinema Ticket Booking

Mobile app design for booking movie tickets without queueing. Includes user flow for film browsing, booking, and e-payment.

![Flutix preview](/images/uiux/flutix.png)

### 7. SIREK – Open Recruitment System for BEM Unsoed

A web-based portal for committee recruitment with registration forms, announcements, and admin dashboard, designed using Laravel + Inertia.js + Vue.

![SIREK preview](/images/uiux/sirek.jpg)

### 8. Disney Website

 A website dedicated to Disney movies. After selecting a specific movie, users can view its synopsis and a list of characters.
![SIREK preview](/images/uiux/disney.jpg)
